# Back up DTR
- Backups do not cause downtime for DTR.
# Perform DTR backup
- You should always create backups from the same DTR replica, to ensure a smoother restore. 
- If you have not previously performed a backup, the web interface displays a warning:

To create a DTR backup, perform the following steps:
1. Run DTR Backup command
2. Back up DTR image content
3. Back up DTR metadata
4. Verify your backup
